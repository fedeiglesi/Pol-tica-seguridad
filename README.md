<h1>Política de Seguridad</h1>
<p><strong>Última actualización:</strong> 16 de diciembre de 2025</p>

<h2>1. Introducción</h2>
<p>Esta Política de Seguridad describe las medidas implementadas para proteger la información procesada por nuestra aplicación, en cumplimiento con las políticas de Meta Platforms, Inc.</p>

<h2>2. Alcance</h2>
<p>Aplica a los datos procesados mediante APIs de Meta (WhatsApp, Instagram, Facebook) y a la infraestructura asociada.</p>

<h2>3. Medidas de Seguridad</h2>
<ul>
    <li>Comunicación cifrada mediante HTTPS (TLS).</li>
    <li>Acceso restringido a sistemas y datos.</li>
    <li>Protección de credenciales y tokens.</li>
    <li>Monitoreo y registros de actividad.</li>
</ul>

<h2>4. Uso de Datos</h2>
<p>Los datos se utilizan exclusivamente para la funcionalidad declarada y no se comparten con terceros no autorizados.</p>

<h2>5. Retención de Datos</h2>
<p>Los datos se conservan solo el tiempo necesario y pueden ser eliminados a solicitud del usuario.</p>

<h2>6. Incidentes de Seguridad</h2>
<p>Ante incidentes, se aplican protocolos internos de mitigación y notificación.</p>

<h2>7. Contacto</h2>
